General Instructions

  • All the bugs which you find should be reported on Github Issues Page of the respective software. Also make sure that you tag the issue as a bug before you raise an issue on the Issues Page.
  • You are expected to test each of the software on both Windows and Linux. Make sure to include on which OS you encountered the bug when you raise an issue on Github.
  • You can also use Github Issues Page of the respective software to request and feature needed in the software. Make sure to tag it as enhancements.
  • In case of any query or doubt you can reach us on Discord.
